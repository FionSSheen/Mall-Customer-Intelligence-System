# Mall-Customer-Intelligence-System
Mall Customer Intelligence System
Overview:

A professional Mall Customer Segmentation and Customer Intelligence Dashboard built using Machine Learning and Streamlit.

The system classifies customers using K-Means Clustering and provides personalized offers, customer analytics, business insights, and profit analysis.
